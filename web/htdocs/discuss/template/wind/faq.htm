注冊與會員登錄
關于注冊
您需要 注冊成為會員 才能正常使用論壇的各項功能，比如︰
  • 發布一個主題
  • 回復其他會員的帖子
  • 編輯自己的帖子
  • 查看其他會員信息,排行榜等
  • 給其他會員發送短信息
  • 搜索您所需要的資料
    等等 .....注冊時，您需要一個用戶名、密碼和一個有效的email地址。輸入您的email地址不會給您帶來'垃圾郵件'，您可以在注冊時選擇隱藏email地址。
    • 關于登錄論壇
    如果您以前是該論壇的會員，請點擊 登錄 進入登錄界面，填寫用戶名和密碼。
    同時您可以選擇是否隱身登錄，如果管理員允許您所處的用戶組擁有此權利，這樣別人將無法知道您是否在線。
    Cookie 有效期介紹: 主要用于在您離開一會兒或者重新進入一個主題或者帖子的時候確認您的身份。當您登錄時， 您將有一個'cookie 有效期:'選項，您可以選擇cookie保存的時間，這時系統將您的用戶名和密碼用cookie保存，儲存在您的電腦。如果您是使用一台共用的電腦，比如在圖書館、網吧，或者其他不能完全信任的地方， 建議使用'即時'選項，以免您的帳號被他人盜用。
    找回密碼
    如果您之前是會員而忘記了密碼，可用此功能 找回密碼(郵箱必須有效)，在取回密碼頁面輸入您注冊時候的用戶名，系統將自動發送密碼到您的有效郵箱中。
    會員控制面板
    關于會員核心數據
    在 編輯會員核心數據 時，考慮到安全性必須輸入原密碼，同時您也可以更改現有的密碼，同時確認您的email地址是有效的，必要時系統將會發送一些論壇信息至您的郵箱，比如密碼發送等。注意不能更改用戶名，如有特殊情況需要修改用戶名，請與管理員聯系。
    關于個性化簽名
    注冊時，可以設置自己的個性簽名。您的簽名將出現在每個帖子底部，您也可以選擇不顯示個性簽名，在發帖和回復的時候不選中"使用簽名"選項。在編輯個性化簽名時您可以使用 Wind Code 進行編輯多彩的簽名。注意簽名的字節數是跟您的等級相關的，所以您的簽名長度是受限制的!
    關于自定義頭像
    個人頭像會出現在每個帖子中用戶名的下方，頭像可以是由論壇提供或者您自己輸入完整的 URL 或上傳 均可以，您可以選擇一個最能代表您個性的頭像。如果論壇允許使用自定義會員頭像， 您可以使用URL 地址的頭像或自己上傳頭像。
    論壇發帖與相關權限
    關于會員{$db_creditname}
    該積分用以判定一個會員對論壇宣傳上的{$db_creditname}! 如果您使用 論壇的URL ?a= 自己的論壇數字ID 給您的一位朋友，使他們登錄到論壇里，您的{$db_creditname}將增加 1 單位。舉例: http://www.phpwind.net/?u=3558 論壇數字ID 可在 控制面板 --> 查看個人資料 里查閱
    關于會員等級提升
    PHPWind 論壇系統提供了一套靈活的等級提升方案，可以根據綜合積分來劃分用戶等級，設定一個好的等級提升方案能更好的體現論壇等級系統的價值

    綜合積分︰綜合積分是按照會員組提升方案中設定的積分比率來計算的，可以根據論壇積分狀況統計合理的設置會員組提升方案
    計算方法︰各項積分*比率的和為綜合積分的值

    如等級提升方案設置
    發貼數 : 1
    威望    : 0.5
    金錢    : -1
    其他    : 0

    綜合積分計算方法如下︰
    發貼數*1 + 威望*0.5 + 金錢*(-1)
    關于論壇權限
    如果您看到論壇的某些版塊為認證版塊，說明您現在的用戶組還沒有權限進入。
    關于發帖的 Wind Code
    可以使用Wind Code 編輯您的帖子、短消息、簽名，您可以設置帖子文字的字體、顏色等格式。Wind Code 代碼使用簡單。
    關于投票
    這項功能允許您提出一個問題，並設置一些可能的答案。其他會員則選答案參與投票，投票結果將在主題上顯示。投票的例子見︰
      什麼是您最喜歡的城市? ( 在編輯框中 以 換行的 形式寫入以下各項 )
    1. 溫州
    2. 杭州
    3. 北京
    4. 上海
    5. 深圳
    　其他會員可以參與投票，您還可以設置投票是否允許多選，及多選的個數。每個會員只有一次投票機會，投票後可編輯投票！
    關于附件
    如果論壇允許使用上傳附件的功能， 您可以在帖子中上傳一些文件.。可以是一個圖片、一個文本、一個壓縮文件等. 您上傳的每種文件都有大小限制( 最大為: 2 M )，論壇跟您的硬盤所采用的文件擴展名並不一致!上傳一個附件，只有下載到自己的電腦上才能瀏覽。論壇限定了上傳文件的類型:可以上傳的文件擴展名: $db_uploadfiletype 發帖完畢後，附件將顯示在您所發布的信息中.
    編輯與刪除帖子
    注冊會員可以編輯或刪除自己的帖子，點按鈕來管理帖子。管理員可以編輯和刪除論壇的所有帖子，版主只能編輯和刪除指定版塊內的帖子。
    關于版主
    版主負責論壇中某個版面。可以修改，刪除或移動他們所管理的版面中的任何帖子等權限。任命某個版面的版主，通常是對本版有特殊貢獻或對某領域有特殊才華的會員的一種獎勵。
    WAP 說明
    
$db_bbsname 目前已開通支持 WAP (手機移動論壇)功能.

手機訪問地址︰$db_bbsurl/wap/

說明︰只要您開通了移動GPRS或聯通CDMA的數據服務，在支持WAP的手機上即可通過此網址訪問 $db_bbsname

手機資費說明︰移動GPRS或聯通CDMA的數據服務包月或流量費用，無需任何額外的費用支出

主要應用範圍︰

    通過手機您可以隨時隨地的瀏覽您在網絡論壇上的活動信息，了解您發表的帖子的閱讀、回復情況。
