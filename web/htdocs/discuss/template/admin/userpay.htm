網上支付功能說明︰
1. PHPWind 網上支付功能同時支持“貝寶支付”，“支付寶支付”，“快錢支付”三種支付手段
2. 開通PHPWind 網上支付功能只需擁有一個“貝寶”帳號或一個“支付寶”帳號或一個“快錢”帳號，並完成相關設置即可，不需要支付任何費用

使用網上支付相關說明︰
1. 開啟網上支付功能即被視為接受“PHPWind”相關協議，如有任何疑義，請關閉此功能
2. 支付過程出現問題請及時和PHPWind 官方取得聯系
3. 因使用網站自身安全原因引起的非法支付PHPWind 不負任何責任


（一）開通快錢支付步驟︰

 1. 注冊一個“快錢”帳號，注冊快錢帳號
2. 設置快錢密鑰，登錄快錢網站︰進入 我的快錢 -> 支付網關 設置您的快錢密鑰，快錢密鑰是包含字母或數字的16位字符串
3. 將您的“快錢商戶號”和“快錢密鑰”填入下面相應快錢信息的文本框中，提交後即可開始使用PHPWind 網上支付功能
4. 由您通過快錢支付產生的相應費用請以快錢網站為準

    獲取快錢商戶號和快錢密鑰︰登錄快錢網站︰進入 我的快錢 -> 支付網關 獲得相應的快錢商戶號和密鑰


（二）開通貝寶支付步驟︰

1. 注冊一個“貝寶”帳號，注冊貝寶帳號
2. 將“貝寶”帳號填寫到論壇後台
    進入論壇後台 -> 網上支付設置 -> 網上支付 -> 貝寶信息 將您的貝寶帳號填寫在“貝寶帳號”欄，提交！
3. 開啟貝寶“即時付款通知”
    登錄貝寶網站︰進入 我的貝寶 -> 用戶信息 -> 即時付款通知習慣設定 -> 編輯 開啟“即時付款通知”並填寫“通知URL”
    通知URL是 您的論壇網址 + '/paypal.php?verifycode=' + 貝寶公共密鑰
    格式如︰http://www.phpwind.net/paypal.php?verifycode=貝寶公共密鑰
4. 完成以上步驟後即可開始使用PHPWind 網上支付功能

    獲取貝寶公共密鑰︰進入 論壇後台 -> 網上支付設置 -> 網上支付 -> 貝寶信息 -> 貝寶公共密鑰
    注︰貝寶公共密鑰是系統為您的網站自動生成的，請不要隨意改動


（三）開通支付寶支付步驟︰

1. 注冊一個“支付寶”帳號,登錄支付寶網站，進入商家工具
2. 設置安全校驗碼，設置成功後您會得到一個安全校驗碼，安全校驗碼一般是一個長度為32的字符串
3. 設置通知URL，通知URL是您的論壇網址 + '/alipay.php'，格式如︰http://www.phpwind.net/alipay.php
4. 將您的“支付寶”帳號和“安全校驗碼”填入下面相應支付寶信息的文本框中，提交後即可開始使用PHPWind 網上支付功能
