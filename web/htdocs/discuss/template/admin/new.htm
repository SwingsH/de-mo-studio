提示信息
1. 使用 PHPWind首頁調用 功能前需要先在 new.php 文件中設置好相關參數，文件里附有說明。
2. 首頁調用方式分為 “論壇版塊調用”，“論壇公告調用”，“論壇基本信息調用”，“論壇會員排行調用”，“論壇文章調用”。
3. 每種調用方式都提供了一系列的參數設置，用戶可根據需要自定義設置。

論壇信息調用演示

可選參數和參數說明︰
\$pre : 前綴,可選參數 : 0.<li>，1.◇，2.•，3.○，4.●，5.-，6.□-
\$member : 是否顯示論壇會員總數和最新注冊會員，可選參數︰0，1
\$article : 是否顯示主題數和帖子數，可選參數︰0，1
\$yesterday : 是否顯示今日，昨日，最高日發帖，可選參數︰0，1
\$online : 是否顯示在線信息，可選參數︰0，1

會員排行調用演示

可選參數和參數說明︰
\$pre : 前綴,可選參數 : 0.<li>，1.◇，2.•，3.○，4.●，5.-，6.□-

論壇文章調用演示

可選參數和參數說明︰
\$pre : 前綴,可選參數 : 0.<li>，1.◇，2.•，3.○，4.●，5.-，6.□-
\$num : 調用主題個數
\$length : 文章標題長度
\$order : 排序方式，可選參數 : 1.最後發表日期 2.主題發表日期 3.回復數 4.查看數
\$fidin : 調用文章的版塊ID，格式︰1_2_3
\$fidout : 不調用的版塊ID，格式︰1_2_3
\$digest : 調用精華帖，可選參數︰0，1
\$postdate : 是否顯示文章發表日期，可選參數︰0，1
\$author : 是否顯示文章作者，可選參數︰0，1
\$replies : 是否顯示回復數，可選參數︰0，1
\$hits : 是否顯示查看數，可選參數︰0，1
\$fname : 是否顯示版塊名稱，可選參數︰0，1
