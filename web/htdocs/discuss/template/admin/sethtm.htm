提示信息
功能說明:
此功能把論壇的URL轉換成更容易被搜索引擎收錄的形式，普通用戶只需開啟靜態目錄部署功能即可，URL形式如︰http://www.phpwind.net/thread.php?fid-2.html，不過這種形式中包含有(.php?), 可能會被某些搜索引擎判定為是動態頁面, 而降低網站內容被收錄的機率，為了能更好的提高您的網站被搜索引擎收錄的機率，可以使用參照下面的方法對服務器進行相關設置

使用方法︰
以apache服務器為例介紹使用方法.

1.在靜態目錄部署中，開啟靜態目錄部署功能，將靜態目錄設置為"-htm-"，將靜態目錄擴展名設置為:".html"
   默認的URL形式為 "http://www.phpwind.net/thread.php?fid=2"
   新的URL形式為    "http://www.phpwind.net/thread-htm-fid-2.html"
   這種純靜態形式的URL更易被搜索引擎收錄

2.Apache Web Server 的配置
   首先確定加載了 Mod_rewrite 模塊，方法: 檢查 conf/httpd.conf 中是否存在以下兩段代碼 (具體路徑可能會有所不同，但形式基本是一樣的):

(一)LoadModule rewrite_module libexec/mod_rewrite.so
(二)AddModule mod_rewrite.c

如果存在，那麼在配置文件（通常就是conf/httpd.conf）中加入如下代碼。此時須注意，如果網站使用通過虛擬主機來定義，請務必加到虛擬主機配置中去，否則可能無法使用。改好後然後將 Apache 重啟.

<IfModule mod_rewrite.c>
RewriteEngine On
RewriteRule ^(.*)-htm-(.*)$ $1.php?$2
</IfModule>
