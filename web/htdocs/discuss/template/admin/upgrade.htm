會員組提升方案

    PHPWind 論壇系統提供了一套靈活的等級提升方案，可以根據綜合積分來劃分用戶等級，設定一個好的等級提升方案能更好的體現論壇等級系統的價值

    綜合積分︰綜合積分是按照會員組提升方案中設定的積分比率來計算的，可以根據論壇積分狀況統計合理的設置會員組提升方案
    計算方法︰各項積分*比率的和為綜合積分的值

    如等級提升方案設置
    發貼數 : 1
    威望    : 0.5
    金錢    : -1
    其他    : 0

    綜合積分計算方法如下︰
    發貼數*1 + 威望*0.5 + 金錢*(-1)
