系統介紹︰

    “論壇營銷計劃”利用 PHPWind 平台把傳統營銷概念與論壇強大互動性相結合，充分挖掘論壇潛在價值，
解決網站經濟來源問題，為站長解除後顧之憂。
    一方面借助 PHPWind 品牌優勢，吸引廣大商家的加盟，解決廣告來源問題；一方面 PHPWind 擁有強大
的用戶基礎，論壇類型幾乎覆蓋所有行業，用戶群集中，能滿足廣大商家的需求；自 PHPWind 5.0版本起，
系統後台集成了“論壇營銷計劃” 功能，站長只需要在後台進行相關設置即可加入“論壇營銷計劃”。


功能說明︰

    區別于傳統營銷系統，站長無須通過申請廣告，獲取代碼，投放廣告等煩瑣的過程，“論壇營銷計劃”自
動幫您完成以上步驟。系統會自動為每個網站生成一對密鑰和一個唯一識別碼，做為網站的唯一標識。更多信息
請關注 PHPWind 最新消息。
