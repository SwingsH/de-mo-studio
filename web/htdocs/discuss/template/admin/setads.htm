提示信息

  • 功能說明:
    就是鼓勵會員宣傳網站的功能。只要用戶把您站點的 url 推薦給其他朋友，並引導其他人訪問您的網站。那他的{$db_creditname}數值將會加一點， {$db_creditname}越大越是表明某會員對網站的宣傳力度越大!
    使用方法: 在論壇任意一動態頁面的 url 後加上 a=用戶名 或 u=用戶數字ID
    比如以下方法均正確(用戶名 yuling 數字ID: 3558)
    http://www.phpwind.net/index.php?a=yuling
    http://www.phpwind.net/index.php?u=3558
    http://www.phpwind.net/read.php?fid=7&tid=154&a=yuling
    http://www.phpwind.net/read.php?fid=7&tid=154&u=3558

  • 注意︰需要注冊是指︰會員引導的朋友需要注冊之後， 會員才會增加貢獻值；而勿需注冊則不限制，只要引導其他人訪問該網站，就會增加!
