是否開啟 JS調用
此功能允許在您的網站首頁或其他地方調用論壇的內容
調用方式分為︰ 版塊調用、公告調用、基本信息調用、會員排行調用、文章調用。		 開啟 關閉
緩存更新時間(秒)
由于部分調用查詢比較消耗服務器資源，系統采用了緩存技術實現數據定期更新
建議設置為︰900
允許調用網站域名(留空不限制)
您可以設置允許調用論壇內容的網站域名列表，只有在列表中的域名才能調用論壇信息
多個域名用 "," 分隔域名，不要包含 http:// 或其他非域名內容

其它設置
強制輸入操作理由︰
為了避免由于管理操作上的不透明而引起論壇會員糾紛，管理員可開啟此項功能，強制要求在前台進行管理操作時必須輸入操作原因。		 開啟 關閉
管理操作原因︰
顯示在前台管理操作的操作原因備選框中，操作者可自行輸入或選擇一個操作原因，操作原因將包含在用戶通知消息和後台管理操作記錄中
每行填寫一個操作原因，空行顯示為分隔符“-------”
